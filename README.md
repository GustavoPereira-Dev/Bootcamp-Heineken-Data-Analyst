# Bootcamp-Heineken-Data-Analyst

Conjunto de projetos referente ao Bootcamp de Análise de Dados (Modelagem, SQL e Excel) da DIO em parceria com a Heineken

[Projeto E-Commerce](./E-Commerce/): Modelagem e desenvolvimento de banco de dados no contexto de vendas de produtos na internet (ECommerce) com tabelas como Cliente (Comprador), Estoque, Produto, Vendedor Terceiro, Fornecedor, etc.

[Projeto Oficina](./Oficina//): Modelagem e desenvolvimento de banco de dados no contexto de uma Oficina de mecânica, em que um cliente pede uma vistoria no seu carro e uma equipe faz o desenvolvimento dos serviços e, por sua vez, a OS para o cliente (Ordem de Serviço)

[Dashboard Xbox](./DashboardXbox/): Criação de um Dashboard focado em inscrições do Xbox com base nos conteúdos vistos (Paletas, Tabelas, Imagens, etc.) e uma espécie de separação mais personalizado pelo instrutor, a ABCDE (Assets, Bases, Cálculos, Dashboard e Extras)
